캠핑 갈래? 개인정보 처리방침

시행일: 2026년 5월 23일
'캠핑 갈래?'(이하 "서비스")는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다.

1. 수집하는 개인정보 항목 및 수집 방법

  • 수집 항목: 앱인토스 익명 식별키(Anonymous Key)
  • 수집 방법: 서비스 이용 과정에서 자동 생성·수집
  • 위치 정보: 사용자가 "현재 위치 사용하기" 버튼을 직접 누른 경우에 한해 1회성으로 사용. 서버에 저장하지 않으며 즉시 폐기 후 날씨 조회에만 사용

2. 개인정보의 수집 및 이용 목적

  • 이용자별 즐겨찾기 캠핑장, 최근 본 위치 동기화
  • 서비스 이용 통계 분석 및 품질 개선
  • 위치 정보: 현재 위치 기반 캠핑 날씨 조회 (저장 없음)

3. 개인정보의 보유 및 이용 기간

  • 익명 식별키: 서비스 이용 기간 동안 보관하며, 이용자가 미니앱 연결 끊기 시 즉시 삭제됩니다.
  • 즐겨찾기·최근 본 위치: 이용자의 디바이스(localStorage)에 저장되며, 앱 삭제 또는 데이터 초기화 시 함께 삭제됩니다. 로그인 시 클라우드(Supabase)에도 동기화되며, 연결 끊기 시 함께 삭제됩니다.
  • 위치 정보: 저장하지 않으며 날씨 조회 요청 직후 즉시 폐기됩니다.

4. 개인정보의 제3자 제공

운영자는 이용자의 개인정보를 외부에 제공하지 않습니다. 단, 다음의 외부 데이터 처리는 발생합니다:

  • Open-Meteo (날씨 API): 사용자가 선택한 위치의 위경도만 익명으로 전달. 익명 식별키 등 개인정보는 전달하지 않음
  • 한국관광공사 GoCamping API: 캠핑장 마스터 데이터 정기 동기화 (이용자 정보 전달 없음)
  • Supabase (DB): 이용자 즐겨찾기·동기화 데이터 저장. 운영자 관리하에 있음

5. 개인정보 처리 위탁

서비스 운영을 위해 다음 업체에 일부 개인정보 처리를 위탁합니다:

  • Vercel Inc.: 백엔드 API 호스팅
  • Supabase Inc.: 데이터베이스 호스팅 (리전: 아시아·태평양)

6. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 개인정보 열람·정정·삭제 요청
  • 토스 앱 내 미니앱 연결 끊기를 통한 데이터 삭제
  • 고객센터 이메일을 통한 별도 요청
